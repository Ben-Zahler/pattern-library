# AXA Materials

### Usage

## Install Icons

`npm install @axa-ch/materials`

## Useage

```
import {
  ArrowRightSvg,
  CollapseSvg,
} from '@axa-ch/materials/icons';
```

Native

```js
this.render() {

}
```

React

```js
this.render() {

}
```

## Icon set

## Add Icon

- Move the raw SVG file into `raw-icons`
- The module will optimize and clean the svg and export it as JS string for further usage.
